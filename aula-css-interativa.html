Módulo 2: Box Model

Todo elemento é uma caixa: conteúdo + padding + border + margin.

Escreva uma regra para um card com espaçamento interno de 16px e borda cinza 1px solid #ccc.

}

Módulo 4: Grid

Grid organiza layout em duas dimensões (linhas e colunas).

Digite um CSS para criar 3 colunas iguais com espaçamento de 16px.

Mostrar resposta 
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}
